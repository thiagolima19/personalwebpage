About me

Since I graduated in 2013, I worked in the field of designing physical exercise programs for various populations. I have experience in the weight room as well as being a personal trainer. In addition to my undergraduate degree in Physical Education from UFPE, I am also a specialist in Exercise Physiology from Estacio de Sa University and hold a Master's Degree in Physical Education from the Federal University of Pernambuco.

In recent times, my thirst for exploring new cultures and learning new skills drove me to join an English language exchange program in Ireland. After spending a year and a half on the Emerald Island, I can confidently say that I have become a more resilient person, in addition to having significantly improved my proficiency in English.

I am currently pursuing an old dream of working in the Information Technology field and living in an English-speaking country, specifically Canada. The time has finally come for me to make this dream a reality. In just a few years, I will be joining a group of individuals who rely on technology for both work and leisure.

Cheat Sheets

HTML

<!DOCTYPE html> It informs the browsing about the document type.
<html> </html> Creates a HTML document.
<head> </head> Sets off the title and other informations, such language.
<meta> Defines metadata about HTML document.
<title> </title> Sets off the title of the webpage.
<link> Defines the relationship between the current document and an external resource.
<script> </script> Used to embed a clients-side scripty such JavaScript.
<body> </body> Sets off the visible part of the webpage.
<a> </a> It defines a hyperlink. href=" " is where you put the link address.
<h1-h6> </h1-h6> Creates a headings. h1 is the most important and h6 is the least.
<p> </p> It creats a new paragraph.
<nav> </nav> Defines a set of navigation links.
<br> </br> Inserts a line break.
<div> </div> it defines a section and is used as a container to be styled by CSS.
<span> </span> Is used to format inline content with CSS.
<pre> </pre> Defines preformatted text.
<ul> </ul> Creates an unordered list.
<ol> </ol> Creates an ordered list.
<li> </li> Is used to create each item in the list.
<form> </form> Difines a form.
<table> </table> Creates a table.
<tr> </tr> Defines the table header.
<td> </td> Sets each cell in a row.
<button> </button> Creates a clickable button.
<!-- --> Creates a comment that is not showed on the webpage.
<img src="" alt=""> Is used to embed an image in an HTML page.
<hr> Is used to create a horizontal rule in the webpage.

CSS

#ID selector: it is a unique identification that is attributed to an HTML element. e.g. #id{}
.Class selector: it is an identification that is attributed to an HTML element. e.g. .class{}
* Universal selector: selects all the elements in the HTML page. e.g. *{}
Tag name: the name of the HTML tag is used for styling. e.g. p{}
pseudo selector: is used to define a special state of an element. e.g. :root{}
align-itens: specifies the alignment inside of flex container
animation: shorthand property to all animation property.
animation-delay: delay for start the animation.
animation-direction: direction for the animation.
animation-name: defines the name for the @keyframes animation.
aspect-ratio: specify the aspect ratio for an element.
background: shorthand property to all background property.
background-attachement: sets if the image scrolls together or will be fixed.
background-color: sets a color to the background.
background-image: sets a image background.
background-repeat: sets if it will be repeat.
background-size: sets the size background image.
border: shorthand property to all borders.
border-color: sets the border color.
border-radius: shorthand for all four border radius.
border-style: sets the border style.
color: sets the text color.
columns: shorthand for column-width and column-count.
cursor: sets the mouse cursor when is pointing over an element.
display: sets how a certain HTML element should be displayed.
flex: shorthand for flex-grow, flex-shrink, flax-basis.
font: shorthand for font properties.
grid: A shorthand for the grid property.
height: sets the height of an element.
justify-content: align items inside a flexible container.
justify-items: align items on a grid container.
@keyframes: specifies the animation code.
margin: shorthand for all margins properties.
overflow: sets what happens to an element when its content goes out of limits.
padding: shorthand for all padding properties.
scroll-behaviour: specifies whether goes straight to the section or in a smooth way.
text-align: specifies the horizontal text alignment.
text-decoration: specifies the text decoration.
transform: applies a 2D or 3D transformation to an element.
transition: shorthand for all transition properties.
translate: specifies the position of an element.
width: specifies the width of an element.
z-index sets the stack order of a positioned element.
